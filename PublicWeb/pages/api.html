Remarks

A promise request will redirect you to a unuique URL that will only reveal the requested numbers after the specified time. Once this specified time has passed, the numbers will always be available from the unuique URL immutably.

PREMIUM

32 bytes of random bits directly from device
rndm.nu/binary64.live
premium only
32 bytes of random bits from specific device type
IDQ20MC1.devices.rndm.nu/binary64.live
premium only

Remarks

.live will return a list of random bytes produced directly on the entropy device for each indiviual http request. A traffic manager ensures the closest rndm.nu entropy device to the client is used. The response will contain additional http headers

Specific devices types are IDQ20MC1 and PQ4000KSI. A traffic manager will ensure the closest entropy device of that type to the client is used.
