PREMIUM

rndm.nu/binary64.live
premium only
16 bytes of signed random bits directly from device
rndm.nu/binary64.signed.live
premium only
32 bytes of random bits from specific device
rndm1.devices.rndm.nu/binary64.live
premium only

Remarks

.live will return a list of random bytes produced directly on the entropy device for each indiviual http request. A traffic manager ensures the closest rndm.nu entropy device to the client is used. The response will contain additional http headers

.signed will return a list of random bytes signed by a EV SSL certificate. The signing is performed on the source entropy device itself and ensures the bytes were not tampered with since leaving the device.
